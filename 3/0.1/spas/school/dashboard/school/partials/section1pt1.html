NOTE - rough priorities and progress:

  1. Functionality of the framework (currently ~60%)
  2. Functionality of each field type (currently varies - average is ~40%)
  3. Style of the framework (currently ~60%)
  4. Style of each field type (currently varies - average is ~10%)
  5. Setting up structure according to AR's layout (currently 0%)
  6. Setting up each field according to AR's layout (currently 0%)
  7. Atypical field tweaks (currently 0%)

NOTE - each field has a type. These are called 'field types' or 'widget types'. There will probably be about a dozen widget types. Currently we have:

  • rating
  • date
  • time
  • text
  • selectFromFew
  • selectFromMany
  • slider
